Anuncio de Actualización para MCT

En julio se actualizarán las variables y scripts. A continuación, te presentamos los puntos clave para asegurar una transición fluida.

1. Instalar Nuevo QGIS

Para garantizar el correcto funcionamiento, se debe descargar la nueva versión estable 3.40.9 LTR de QGIS.

Descargar QGIS 3.40.9 LTR

2. Sincronizar Capas

Se recomienda usar OneDrive para recibir automáticamente las actualizaciones de las capas de información y de los scripts.

3. Frecuencia

  • Las capas se actualizan cada 3 meses.
  • Actualizaciones urgentes pueden solicitarse por correo electrónico.

Variables Actualizadas en Julio

La actualización de julio incluye cambios en las siguientes capas de información:

  • Sitios arqueológicos
  • Propiedad fiscal

  • Glaciares

  • Sitios Prioritarios de Conservación
  • Humedales urbanos
  • Comunidades Indígenas

  • Espacios Costeros Marinos Protegidos para Pueblos Indígenas (ECMPO)
  • Monumentos Históricos
  • Líneas de transmisión eléctrica

Alternativa Manual

Si no usas OneDrive, puedes descargar las variables y el código desde la carpeta de SharePoint, en el portal de estudios territoriales.
